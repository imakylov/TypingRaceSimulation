# TypingRaceSimulator

Object Oriented Programming Project — ECS414U

## Description

A simplified simulation of a typing race build with java to run in a terminal with mechanics of burnout when a typist is typing too fast, mistyping and finishing a race.

## Running

To run the typing race simulator simply run the Main.java file.

## Testing

To check if all of the relevant methods in Typist.java work properly run TypistTests.java
